🎨 Theme Synchronization Test

This test verifies that the popup respects the light/dark mode setting from the options page.

🔧 Setup Instructions

1. Load Extension
Load the extension in Chrome using Developer Mode:
chrome://extensions/ → Enable Developer Mode → Load unpacked
2. Open Options Page
Right-click extension icon → Options
OR navigate to: chrome-extension://[extension-id]/options/options.html

🧪 Test Cases

Test 1: Default Theme
1. Fresh install - theme should follow system preference
2. Open popup and verify it matches system dark/light mode
✅ Expected: Popup theme matches system preference
Test 2: Manual Light Mode
1. In options page, click the dark mode toggle to set light mode
2. Open popup immediately after
✅ Expected: Popup shows light theme (white background, dark text)
Test 3: Manual Dark Mode
1. In options page, click the toggle again to set dark mode
2. Open popup immediately after
✅ Expected: Popup shows dark theme (dark background, light text)
Test 4: Text Readability
1. Switch to dark mode in options
2. Open popup and check all text elements are clearly readable
3. Pay attention to: statistics labels, activity text, footer links
✅ Expected: All text has sufficient contrast in dark mode
Test 5: Real-time Sync
1. Keep popup open
2. In another tab, open options page
3. Toggle theme while watching popup
✅ Expected: Popup theme changes instantly without closing/reopening
Test 6: Persistence
1. Set a specific theme in options
2. Close and reopen popup multiple times
3. Restart browser and test popup
✅ Expected: Theme preference persists across sessions

🔍 What to Look For

Light Theme Popup:
• White/light gray backgrounds
• Dark text (nearly black)
• Light borders and surfaces
• Orange/blue accent colors preserved
Dark Theme Popup:
• Dark gray/blue backgrounds
• Light text (nearly white)
• Dark borders with subtle highlights
• Orange/blue accent colors preserved
⚠️ Troubleshooting:
• If popup doesn't change, check browser console for errors
• Verify Chrome storage permissions are granted
• Test with background script enabled/disabled
• Check if system preference override is working

🛠️ Technical Details

Storage Key: themeMode
Values: "light" | "dark" | undefined (system)
CSS Classes: .light-theme | .dark-theme
Sync Method: Chrome storage change listener
